# Google Translate

A google translate http api

[![pdm-managed](https://img.shields.io/badge/pdm-managed-blueviolet)](https://pdm.fming.dev)
</div>

## Install

首先需要安装[pdm](https://github.com/pdm-project/pdm)

安装完成后执行安装依赖
```bash
pdm install
```

## Run

```bash
pdm run start
```
